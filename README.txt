Сначала с помощью nmap сканятся порты 80 и 443. На открытых портах находятся http и https.
На каждом из найденных будет работать ffuf.
Должен быть установлен ffuf(apt install ffuf)
Должен быть установлен nmap(apt install nmap)
Логи сохраняются в csv формате
первый аргумент - ip
второй - wordlist для директорий
третий(необязательный) - wordlist для файлов.
